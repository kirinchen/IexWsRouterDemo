# IexWsRouterDemo

## Init Project
1. Install Nodejs and npm 
2. Go to [Root]/IexTradingDemo/ 
3. type cmd : ``npm install``

## Start Project

### Run Server
* Go to [Root]/IexTradingDemo/ 
* type cmd : ``node server.js``

### Run test client
* Go to [Root]/IexTradingDemo/ 
* type cmd : ``node DemoWsClient.js``
